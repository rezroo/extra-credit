Convert Example Docker Vote Application to Kubernetes

  1. As a kubernetes cluster admin create three PersistenVolumes of type hostPath and size 50Mi, 100Mi, and 150Mi.
  2. As a kubernetes user create one PersistentVolumeClaim with accessMode of read and write of size 80Mi
  3. Check to make sure the PVC is bound to a PV
  4. Create a deployment with 1 replica for image postgres:9.4 that user the PVC created in step 2
  5. Create a Service for the previous deployment which targets port 5432
  6. Create a deployment with 2 replicas for image redis:alpine with port 6379
  7. Create a Service for the previous deployment which targets port 6379
  8. Create a Deployment with 2 replicas for image dockersamples/examplevotingapp_worker
  9. Create a Deployment with 1 replica for image dockersamples/examplevotingapp_vote:before
  10. Create a Service of type NodePort for the previous deployment
  11. Create a Deployment with 1 replica for image dockersamples/examplevotingapp_result:before
  12. Create a Service of type NodePort for the previous deployment
