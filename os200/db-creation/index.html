Convenience Script for MySql Database Creation

You can use this script to quickly create and test your openstack databases.
  1. Click the link and copy the script URL.
  2. In your lab environment "wget <copied URL>"
    ie. wget http://trainer.edu.mirantis.com/extra-credit/os200/db-creation/create-mysql-db-for.sh
  3. chmod a+x ./create-mysql-db-for.sh

Usage: ./create-mysql-db-for &ltdbname> [optional-management-nic=eth0]

Example: ./create-mysql-db-for keystone
