Convenience Script for MySql Database Creation

You can use this script to quickly create and test your openstack databases:

wget http://trainer.edu.mirantis.com/reza/os200/db-creation/create-mysql-db-for.sh

chmod a+x ./create-mysql-db-for.sh

Usage: ./create-mysql-db-for <dbname> [optional-management-nic=eth0]

Example: ./create-mysql-db-for keystone
