Improve NTP configuration to default to local clock:

You can add the local clock to the ntp.conf file so that if the NTP server is disconnected from the internet, NTP server provides time from its local system clock.

vi +15 /etc/ntp.conf

Insert the following two lines:

server 127.127.1.0 # local clock

fudge 127.127.1.0 stratum 10

Execute the command to restart ntp:

service ntp restart

or

sysctl reload ntp.service

ntpq --numeric --peers

For additional documentation see:

http://support.ntp.org/bin/view/Support/UndisciplinedLocalClock
